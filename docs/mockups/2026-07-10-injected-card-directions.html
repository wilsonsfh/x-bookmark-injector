Card anatomy (both directions carry the same data)

  • 📌 From your bookmarks — provenance label so you never confuse it with the real feed.
  • Saved #12 of 87 — chronological save rank (order #k of N). Order is reliable; it's the position in X's save-ordered list. "12th oldest" framing supports draining the backlog.
  • 74 left — bookmarks not yet cleared (total − Done'd).
  • posted Jun 21, 2026 — the tweet's real posted timestamp (from the API).
  • No exact "bookmarked-at" clock time for old items — X doesn't return it. New Done/Keep actions get a local timestamp going forward.
  • Done (remove from X) → real DeleteBookmark mutation · Keep for later → local dismiss, stays in X.
